## Generation Pipeline

```
note source: ~/repos/note/{key}.md   (private; Copper's full note)
      ↓ frontmatter `publish: true` + `publish_to: textbook-notes`
      ↓ textbook-share-sync.py (launchd q15m)
      ↓ extracts `## KEY TAKEAWAYS` + `## TEACHING SLIDES` only
      ↓ writes shareable derivative
      ↓
this repo: ~/repos/note/textbook-notes/{takeaways,slides,cards}/{book}/{ch}.{md,png}
      ↓ Marp CLI → .html / .png slides
      ↓
SHARE TARGET (Phase-9d 2026-04-25 onwards):
   primary  → 診所官網 (sub-project under boan-emr/) — current public face
   legacy   → copper0722/textbook-notes GH Pages — kept as mirror; may retire
```

**Privacy gate**: only KEY TAKEAWAYS + TEACHING SLIDES leave the private `note/` parent. The full note (with copyrighted excerpts, work-in-progress reasoning) stays in `~/repos/note/{key}.md` (PRIVATE).
