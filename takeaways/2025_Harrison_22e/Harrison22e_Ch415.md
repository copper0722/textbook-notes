# 糖尿病：診斷、分類與病理生理｜Diabetes Mellitus: Diagnosis, Classification, and Pathophysiology

## KEY TAKEAWAYS

- **定義**：DM = metabolic disorders 共同 phenotype **hyperglycemia**；因**insulin secretion + action + glucose production** 異常組合所致
- **兩大 categories + 多種 other**：Type 1 (autoimmune)、Type 2 (resistance + secretion)、MODY (monogenic)、neonatal、pancreatogenic、endocrinopathy、drug-induced、GDM
- **Type 1 DM staging**：Stage 1 (≥2 ICA + normoglycemia) → Stage 2 (dysglycemia) → Stage 3 (hyperglycemia, clinical DM)
- **Teplizumab** 為首個 FDA-approved disease-modifying therapy for Stage 2 Type 1 DM (delays Stage 3 median 32.5 months)
- **HLA**：~50% Type 1 DM 遺傳風險；DR3-DQ2 + DR4-DQ8 預期；protective DRB1*1501
- **Type 2 DM genetic**：polygenic、>600 loci、<10% 遺傳風險解釋；TCF7L2 最 prominent
- **全球 537M DM (2021) → 預測 2045 783M**；台灣 ~3M；**75% 在 LMIC**
- **診斷**：HbA1c ≥6.5、FPG ≥126、OGTT 2h ≥200、random ≥200 + 症狀；**異常 repeat 確認**
- **Prediabetes**：A1c 5.7-6.4、IFG 100-125、IGT 140-199；**DPP lifestyle ↓58%、metformin ↓31% 進展**
- **Insulin secretion molecular cascade**：glucose uptake (GLUT) → glucokinase → ATP → KATP closure → Ca²⁺ influx → insulin release；**SUR1 = sulfonylurea binding site**
- **Incretin (GLP-1, GIP)**：enhance glucose-stimulated insulin + suppress glucagon；基礎現代 T2DM 治療
- **Type 1 DM 病理**：autoimmune、**CD8+ T cell cytotoxicity** primary；ICAs diagnostic + prognostic marker
- **Type 2 DM 病理**：insulin resistance 早於 secretion defect；**obesity 80%**；**postreceptor defects + adipokine abnormalities + glucotoxicity + lipotoxicity**
- **MASLD** (metabolic dysfunction-associated steatotic liver disease) 常併發 Type 2 DM
- **Monogenic DM**：<5% Type 2 DM；HNF-1α (sulfonylurea-responsive)、glucokinase (mild stable)、HNF-1β (renal cysts)
- **Neonatal DM (<6 mo)**：Kir6.2/ABCC8 activating mutations **sulfonylurea-responsive** → 可停 insulin
- **Classification 個別困難**：atypical cases 增；ICA + C-peptide + genetic testing 輔助
