# 糖尿病：管理與治療｜Diabetes Mellitus: Management and Therapies

## KEY TAKEAWAYS

- **Goals**：消除 hyperglycemia symptoms、降併發症、維持 normal lifestyle
- **Multidisciplinary team** essential
- **HbA1c target 個別化**：大多 <7%、老年 <7.5-8%；TIR >70% 為 CGM 目標
- **DSMES 持續過程**，非一次完成
- **MNT Mediterranean pattern + 限 carb + 避免 added sugar**
- **Exercise 150 min/wk** moderate aerobic + resistance + flexibility
- **CGM 為 type 1 + insulin-using type 2 DM 標準**
- **HbA1c 2-4x/yr**；eAG 轉換表
- **Type 1 DM：Intensive insulin therapy + CSII/AID 為標準**
- **Teplizumab** 首個 disease-modifying for type 1 DM stage 2 prevention
- **Type 2 DM Paradigm shift**：**SGLT2i + GLP-1 RA 基於 CV/renal benefit**，不再 stepwise metformin-then-others
- **Metformin first-line** 仍；但 CVD/HF/CKD/obesity → SGLT2i 或 GLP-1 RA 並列 first-line
- **GLP-1/GIP tirzepatide** 提供最強 weight loss + 血糖
- **DKA**：hyperglycemia + ketones + anion gap acidosis；**insulin + fluid + K replacement**
- **HHS**：hyperosmolar + dehydration + minimal ketones；elderly type 2 DM
- **Hypoglycemia < 70**；glucagon 必備 for type 1 DM
- **Hospitalized DM**：target 140-180；avoid hypo
- **Pregnancy**：preconception HbA1c <6.5; switch to insulin
- **Technology 快速進化**：hybrid closed-loop → fully closed-loop "bionic pancreas"、stem cell beta cell replacement
