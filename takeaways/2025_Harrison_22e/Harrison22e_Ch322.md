# 慢性腎臟病｜Chronic Kidney Disease

## KEY TAKEAWAYS

- **定義**：腎臟結構/功能異常 ≥3 個月；ESRD 正式改名 stage 5 CKD
- **KDIGO 雙軸分期**：GFR + UACR；風險 continuous；**KFR equation** 輔助預測
- **病理生理雙機制**：(a) **specific initiating**（遺傳/免疫/代謝/毒素），(b) **nonspecific hyperfiltration + hypertrophy + RAAS + TGF 失控**
- **五大 etiology**：DM nephropathy 北美第一，Taiwan 亦主導 (~45%)；GN、HTN-associated、ADPKD、tubulointerstitial
- **基因** 占 20% adult-onset；APOL1 西非族裔關鍵；**inaxaplin** 2023 首個 genome-guided 藥物
- **尿毒症三 sphere**：毒素累積、內分泌/代謝 loss、**systemic inflammation (MIAC syndrome)**
- **CKD-MBD cascade**：P ↑ → FGF-23 ↑ → calcitriol ↓ → secondary HPT；late → adynamic bone + vascular calcification + **calciphylaxis**
- **心血管 mortality 每 stage ↑ 10-200x**；HTN、HF (HFrEF + HFpEF)、LVH、**low-pressure pulmonary edema (uremic-specific)**、pericarditis
- **Anemia of CKD**：EPO + iron + inflammation；**Target Hb 100-115** (不 normalize)；**HIF-PHI (roxadustat) 新 class**
- **Bleeding-thrombosis paradox**：**DDAVP + cryoprecipitate + tranexamic acid** 急救；**apixaban** 替代 warfarin 因 calciphylaxis
- **現代 CKD 三聯處方**：**ACEi/ARB + SGLT2i + finerenone**（± semaglutide）
- **BP target <130/80**；RAASi **<30% eGFR drop = 可接受**；eGFR <30 continuation 降 CV/mortality
- **HK management revolution**：**patiromer + SZC** 允許 RAASi + MRA 續用
- **Acidosis HCO3 <20-22**：NaHCO3 降 progression + protein catabolism
- **PEW 評估**：SGA + edema-free weight + BIA/DXA
- **Nephrotoxin avoidance**：NSAIDs、iodinated/gadolinium contrast 謹慎
- **Dialysis initiation**：symptom-driven > eGFR cutoff；**IDEAL trial** 否定 early start
- **Transplant**：best rehab option；**pre-emptive transplant** (from living donor) > dialysis-then-transplant
- **Global burden**：2040 CKD YLL 將達 5th globally；發展中國家尤劇
